Порядок развертывания:

1. Создать новую директорию
2. Положить в нее проекты lunch-backend, lunch-frontend и файл docker-compose.yml
    Структура проекта:
    - <Созданная_директория>
        |__ lunch_backend
        |__ lunch-frontend
        |__ docker-compose.yml
3. Открыть терминал в созданной директории и прописать команды "docker-compose build" и "docker-compose up -d"
